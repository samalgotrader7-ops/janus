# Janus Tutorials

Four progressive walkthroughs for new users. Each is one
screenful — read them in order.

| # | Tutorial | What you'll learn |
|---|---|---|
| 01 | [Hello, Janus](01-hello-janus.md) | Install, configure a model, first turn |
| 02 | [Your First Skill](02-your-first-skill.md) | Write a skill, trigger auto-load, promote |
| 03 | [Memory Loop](03-memory-loop.md) | Memory proposals, review flow, /memory commands |
| 04 | [Connect MCP](04-connect-mcp.md) | Configure + connect MCP server, call its tools |

After these, see:

- [`scripts/install_services.sh`](../scripts/install_services.sh) — production VPS deploy with systemd auto-restart
- [`docker-compose.yml`](../docker-compose.yml) — Docker stack (web + telegram + daemon)
